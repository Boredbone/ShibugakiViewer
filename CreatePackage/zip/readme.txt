ShibugakiViewer


・概要
画像ファイルの管理・表示アプリです。
登録されたフォルダ内の画像ファイルをデータベース化し、画像のサイズ・作成日時・タグ・評価などの条件を指定して検索することができます。
"AND/OR"を使用した詳細な条件の設定が可能です。
PNG,JPEG,BMP,GIF,WEBP,WMF,EMF,TIFF形式の画像に対応しています。
ベクタ形式画像(WMF,EMF)を、品質を保ったまま拡大して表示することができます。


・動作環境
Windows 10で操作するデスクトップアプリです。
.NET Core Desktop Runtime 3.1以上が必要です。


・インストール・アンインストール
同梱のShibugakiViewerInstaller.exeからインストールしてください。
アンインストールは、Windowsのコントロールパネルから行ってください。
